# Shannon’s Activist Toolkit: Women’s Education

This website is my personal project, created as part of **Girls Who Code** to raise awareness and provide resources for **women’s education**. The goal is to inspire action, share helpful tools, and make it easy for anyone to stand up for this cause.

---

### What My Website Does
- Built a **multipage website** using HTML, CSS, and JavaScript.  
- Provides interactive elements like buttons and menus to guide users through resources and actions.  
- Uses a **custom design** with colors, images, and fonts that reflect the theme of women’s education and empowerment.  
- Includes **alt text for every image** to ensure accessibility.  
- Code is commented to explain how the website works, making it easy to read and maintain.

---

### Ways to Explore or Extend
If someone wants to build on this project, they could:  
- Add more interactive features or resources for other causes.  
- Refactor code to improve readability and performance.  
- Host the website live on **GitHub Pages**.  
- Expand the toolkit with additional pages, graphics, or key features.

## File Overview

### ← README.md
This file! It explains the purpose of the website, what it does, and how it was built.

### ← index.html
Holds all the website content — text, images, buttons, and links.

### ← style.css
Controls the visual style of the site — fonts, colors, layout, and overall design.

### ← script.js
Adds interactivity — handles user actions like clicks, menu navigation.

### ← assets
Contains images and other media files used in the website. Any image added to the site is stored here.
